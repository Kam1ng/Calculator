# 更新日志 (Changelog)

所有重要的变更都会记录在此文件中。

## [最新版本] 2026-06-25

### Bug 修复
- **键盘输入重复问题**
  - 原因：`display.setEditable(true)` 时，JTextField 默认输入行为与 KeyListener 处理冲突
  - 修复：将 `editable` 设为 `false`，完全由自定义逻辑控制输入
  - 影响：现在键盘输入不会再重复

- **一元负号问题**
  - 原因：`sqrt(-1)` 等表达式中的 `-` 被当作二元减法，缺少操作数导致报错
  - 修复：添加一元负号检测，在表达式开头、左括号后、逗号后、运算符后的 `-` 自动补 `0`
  - 影响：现在支持 `sqrt(-1)`、`-5 + 3`、`sin(-pi/2)` 等表达式

### 功能改进
- **对数函数区分**
  - `lg(x)`：常用对数（以10为底）
    - 示例：`lg(100)` → `2.0`
  - `ln(x)`：自然对数（以e为底）
    - 示例：`ln(e)` → `1.0`
  - `log(base, x)`：指定底数的对数（使用逗号分隔底数和参数）
    - 示例：`log(2,8)` → `3.0`
    - 公式：`log_base(x) = ln(x) / ln(base)`

- **详细错误提示**
  - 所有错误类型均提供具体提示信息
  - 错误提示包含具体错误值，便于用户定位问题
  - 新增错误类型：
    - `Empty expression`：空表达式
    - `Invalid expression`：无效表达式
    - `Invalid token: xxx`：无法识别的字符
    - `Mismatched parentheses`：括号不匹配
    - `Invalid comma usage`：逗号误用
    - `Not enough operands for xxx`：运算符/函数缺少参数
    - `Division by zero`：除数为0
    - `Cannot sqrt negative number (got -1.0)`：负数开方
    - `Lg argument must be > 0 (got -1.0)`：对数参数≤0
    - `Log base must be > 0 and != 1 (got 1)`：对数底数无效
    - `Asin argument out of range [-1, 1] (got 2)`：反三角函数越界

### 界面调整
- 科学面板新增 `lg` 按钮
- 科学面板新增逗号 `,` 按钮，用于 `log(base, x)` 格式
- 恢复 `deg` 和 `rad` 按钮

### 技术细节
- 表达式解析器升级：支持多参数函数和逗号分隔符
- 词法分析器升级：正确区分 `log`、`ln`、`lg` 三种对数函数
- 错误处理升级：全面检查表达式有效性，提供详细错误信息
- 一元运算符支持：自动检测并处理一元正负号

## [历史版本] 2026-06-24

### 功能特性
- 基本运算：加、减、乘、除、幂运算、小数点
- 科学计算：三角函数、反三角函数、开方
- 数学常数：π (pi)、e
- 表达式解析：完整的中缀表达式解析，支持运算符优先级
- 历史记录：自动保存、按日期分类、查看、清空
- 科学模式：切换显示更多科学计算按钮
- 键盘支持：数字输入、运算符、退格、回车计算
- 菜单功能：复制、粘贴、历史记录、科学模式切换、About对话框
