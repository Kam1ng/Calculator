### 3.4 表达式解析算法

采用**双栈法**（Shunting-yard 算法）进行中缀表达式求值：

```
算法: 中缀表达式转后缀并求值

输入: 中缀表达式字符串
输出: 计算结果

步骤:
1. 初始化操作数栈和运算符栈
2. 从左到右扫描表达式
3. 如果是数字, 入操作数栈
4. 如果是运算符:
   a. 如果运算符栈顶优先级 >= 当前运算符, 出栈计算
   b. 否则, 当前运算符入栈
5. 如果是左括号, 入运算符栈
6. 如果是右括号, 弹出并计算直到遇到左括号
7. 弹出所有剩余运算符并计算
8. 返回操作数栈顶元素
```

### 4.2 核心算法实现

#### 4.2.1 表达式解析器
```java
private double evaluateExpression(String expression) {
    // 预处理: 替换常数
    expression = expression.replace("π", String.valueOf(Math.PI));
    expression = expression.replace("e", String.valueOf(Math.E));
    
    // 双栈法求值
    Stack<Double> numbers = new Stack<>();
    Stack<Character> ops = new Stack<>();
    
    // 逐字符处理
    for (int i = 0; i < expression.length(); i++) {
        char c = expression.charAt(i);
        
        if (Character.isDigit(c) || c == '.') {
            // 读取完整数字
            StringBuilder sb = new StringBuilder();
            while (i < expression.length() && 
                   (Character.isDigit(expression.charAt(i)) || 
                    expression.charAt(i) == '.')) {
                sb.append(expression.charAt(i++));
            }
            i--;  // 回退一个位置
            numbers.push(Double.parseDouble(sb.toString()));
        }
        // ... 处理运算符和括号
    }
    
    return numbers.pop();
}
```

#### 4.2.2 运算符优先级
```java
private int precedence(char op) {
    switch (op) {
        case '+':
        case '-': return 1;
        case '*':
        case '/': return 2;
        case '^': return 3;
        default:  return 0;
    }
}
```

#### 4.2.3 科学函数处理
```java
private double applyFunction(String func, double value) {
    switch (func.toLowerCase()) {
        case "sin": return Math.sin(value);
        case "cos": return Math.cos(value);
        case "tan": return Math.tan(value);
        case "log": return Math.log10(value);
        case "ln":  return Math.log(value);
        case "sqrt": return Math.sqrt(value);
        case "abs": return Math.abs(value);
        default: return value;
    }
}
```

#### 4.3.2 加载历史记录
```java
private void loadHistory() {
    File file = new File("calculator_history.txt");
    if (!file.exists()) return;
    
    history.clear();
    try (BufferedReader reader = new BufferedReader(
            new FileReader(file))) {
        String line;
        while ((line = reader.readLine()) != null) {
            history.add(line);
        }
    } catch (IOException ex) {
        // 异常处理
    }
}
```
